# VIBE Credit Pricing

| Tier | Credits | Examples |
|------|---------|----------|
| Free | 0 | `operations` (discovery), `trial-auth` |
| Light | $0.005 | `swap-quote`, `defi-discover`, `bags/positions` |
| Standard | $0.01 | `data-provider`, `defi-quote`, `bags/claim-fees` |
| Heavy | $0.02 | `swap`, `evm/swap`, `defi/deposit` |
| Premium | $0.05 | `bags/launch-token` |

**Trial:** 30 free requests/day without API key. Read-only scopes (no trading).

**Full access:** Create API key at [vibe.airforce/settings/api-keys](https://www.vibe.airforce/settings/api-keys)
